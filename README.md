# Génesis — Plan de Trabajo APIs

Plan de trabajo interactivo para el seguimiento de contratación de APIs del Proyecto Génesis.
Genomma Lab Internacional — Confidencial — Abril 2026

## Features
- Vista de 86 APIs/servicios agrupados por categoría
- Asignación de equipo, responsable y sponsor por servicio
- Seguimiento de avance (%) y status de contratación
- Filtros por status, equipo y prioridad
- Persistencia local (localStorage)
